## 🛠 Công nghệ sử dụng

- Database: MySQL

## 📂 Cấu trúc thư mục
- `database.sql`: Chứa script tạo bảng, dữ liệu mẫu, View, Procedure và Trigger.
- `main.py`: Mã nguồn Python điều khiển hệ thống.

## 🚀 Cách chạy dự án
1. Chạy file `database.sql` trong MySQL Workbench.
2. Cấu hình thông tin kết nối (user, password) trong file `main.py`.
3. Chạy lệnh `python main.py`.
